Práctica 5.- Uso de Canvas II

1. Usando la Api Canvas, combina arcos y/o círculos con cualquier elemento visto en canvas. Se valorá la originalidad.

Fallo de compatibilidad con el canvas.

2. Usando la Api Canvas, carga una imagen, recortará y redimensionala en un canvas. En el archivo debe aparecer también la imagen original. Se valorá la originalidad.

Fallo de compatibilidad con el canvas.

3. Usando la Api Canvas, crea una figura con stroke, haz un degradado dentro de la figura. Dale un nombre y colócalo debajo de la figura. Se valorá la originalidad.

Fallo de compatibilidad con el canvas.

Opcional: Crea una animación usando imágenes.

Fallo de compatibilidad con el canvas.
